# Action

**Action** is a two-sided web platform for the film industry. It connects pre-union creatives — actors, writers, directors, producers, cinematographers, editors, and composers — with auditions, education, networking, and events. It also serves a separate fan audience with industry news, events, and access to emerging filmmakers.

- **Deployed app:** _coming soon_
- **Business plan:** https://action-web-kappa.vercel.app

## Tech stack at a glance

Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS v4, shadcn/ui, and lucide-react, deployed on Vercel. See [docs/STACK.md](docs/STACK.md) for the full list and reasoning.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
