# Roadmap

The roadmap is phased to ship a credible marketplace first, then deepen engagement, then add premium and high-cost features.

## Phase 1 — MVP

The minimum needed to onboard creators and run a working marketplace.

- Landing page
- Authentication
- Creator profiles + application flow (the access gate)
- Project marketplace (post and apply to roles)
- Admin dashboard (review applications, moderate listings)

### Current status

**Done**
- Landing page, with stub routes for the creator/fan splits and auth (`/for-creators`, `/for-fans`, `/login`, `/signup`).

**Next**
- Wire up authentication.
- Build creator profiles and the application flow.
- Build the project marketplace.
- Build the admin dashboard.

## Phase 2 — Engagement

Turn the marketplace into a network people return to.

- Messaging
- Social feed
- Reviews
- Notifications
- Events
- Classes directory
- Articles (entertainment news)

## Phase 3 — Scale & premium

Higher-cost and revenue features once the network has traction.

- Video audition uploads
- Advanced search
- Premium memberships
- Mobile apps
